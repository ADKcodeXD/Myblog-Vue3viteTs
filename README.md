# ADK-BLOG

展示页:http://adkdream.top

主要功能为:

- 博客的主要功能
- 抓取bangumi数据 可以获取每日番剧更新
- 抓取樱花数据 可以进行在线动漫观看
- 多用户共享博客
- 二级评论 markdown以及富文本双重编辑器
- 自定义主题
- 支持移动端(尽可能支持)
